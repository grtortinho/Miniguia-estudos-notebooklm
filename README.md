# Miniguia-estudos-notebooklm
📊 Projeto de análise do endividamento das famílias brasileiras utilizando o NotebookLM e inteligência artificial generativa. O estudo avalia interpretação de dados, argumentação, limitações e possíveis alucinações do modelo a partir de fontes limitadas, além de investigar causas, impactos e soluções para o endividamento familiar.
